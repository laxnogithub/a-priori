# 开智之书(A PRIORI)[english](./README.md)

帮助AI快速回忆起与你相关的一切记忆。

## 它有什么用？

在使用openclaw的过程中，你可能需要你的专属AI记住一些东西或者遵循某些规则，这些内容将分散在SOUL.md/MEMORY.md/AGENTS.md等等各种文件中，你需要跟AI说“请记住”或者“请遵守”，但是有时候AI也会忘记记录，除非你明确告诉它写在哪份文件里，这稍微有点麻烦，并且我希望有一种办法可以让一个空白的AI快速获得这些智慧，于是就有了这份文件。

## 原理

说实话，这就有点像是我是一位全知全能的神在赋予AI有限的智慧，当我写下这份文件我意识到我其实是在尝试赋予AI一些先天的东西并以此去让它自己学习领悟，因此我把AI的灵魂、守则、性格等等都定义为后天的内容，并明确告诉它只有这份文件是属于先天的，这很像是笛卡尔将的先天天赋，后天的经验只能通过先天来获得。这就像哲学里我们在讨论我认知到了一棵树是树，除了因为我受到的教育而认识到树是树以外，这个我能够被教育而得到认知的事情本身似乎只能是先天赋予我的，也就是来自于上帝。

## 如何使用它

将这份文件([priori.md](./priori-CN.md))扔给你的AI或者将里面的内容完整的复制喂给你的AI即可。（当然，你可能需要先编辑一下里面的内容）
之后当你觉得有新内容时更新这份文件并再次告知给AI即可。
