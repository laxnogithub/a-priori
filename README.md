# The Book of Awakening (A PRIORI)[中文](./readme-CN.md)

Helping AI instantly recall every memory of you.

## What is it for?

When using OpenClaw, you may want your personal AI to remember certain things or follow specific rules. These contents would normally scatter across SOUL.md, MEMORY.md, AGENTS.md, and various other files. You'd have to tell the AI "please remember" or "please follow," but sometimes the AI forgets to record them unless you explicitly tell it which file to write to. That's slightly troublesome. I wanted a way for a blank AI to quickly acquire this wisdom—so this file was born.

## How it works

Honestly, it's a bit like I am an omniscient deity bestowing limited wisdom upon the AI. As I wrote this file, I realized I was essentially trying to give the AI something innate—a priori—and letting it learn and comprehend from there. So I defined the AI's soul, principles, personality, and so on as derived (a posteriori) content, and made it clear that only this file is innate (a priori). This resembles Descartes' notion of innate ideas: derived experience can only be obtained through the innate. It's like the philosophical discussion of how we recognize a tree as a tree—beyond recognizing it through education we received, the very capacity to be educated and gain cognition must itself be innately endowed, that is, from God.

## How to use it

Simply toss this file ([priori.md](./priori.md)) to your AI, or copy its entire contents and feed it to your AI. (Of course, you may want to edit the content first.)
Afterward, when you have new content to add, update this file and inform the AI again.
